[> Just as there is gcc for C and g++ for C++, there is clang for C and clang++ for C++.](https://cplusplus.com/forum/unices/121465/)

# g++ (build & run)
`g++ Main.cpp -o Main`  
`./Main`  

# clang++ (build & run)
`clang++ Main.cpp -o Main`  
`./Main`  

# requirements
**g++**: `sudo apt install g++`  
**clang++**: `sudo apt install clang`
