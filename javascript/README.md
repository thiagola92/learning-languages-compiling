# node (run)
`node Main.js`  

# deno (run)
`deno run Main.js`  

# deno (build & run)
`deno compile Main.js`  

# requirements
**node**: `curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash - && sudo apt-get install -y nodejs`  
**deno**: `curl -fsSL https://deno.land/install.sh | sh`
